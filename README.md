Calculator

This is a simple calculator application built using basic web technologies. It performs standard arithmetic operations like addition, subtraction, multiplication, and division.

Features
- Addition, subtraction, multiplication, division
- Clean and simple user interface
- Responsive design
- Easy to use

Tech Stack
- HTML
- CSS
- JavaScript

Usage
- Enter numbers using buttons
- Select an operation (+, -, *, /)
- Click equals to get the result
- Use clear button to reset
